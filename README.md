# Structr

**Structr** is a CLI tool designed to help users navigate, map, and create complex filesystems.

## 1. Installation
Documentation to come

An easy way to *optimize* efficiency is by creating an alias so you can call the structr navigation
tool using "ccd" "cdd" or something similar. To set this up, run:
`echo 'alias "ccd"="structr"' >> ~/.bashrc`
in your Linux command line.
